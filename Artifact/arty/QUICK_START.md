# Arty - Quick Reference

## Start Services

```powershell
# Discord Bot
npm run discord:start

# LinkedIn Scheduler  
npm run linkedin:start

# Telegram Bot
npm run telegram:start

# Notion Client
npm run notion:start
```

## Configuration Files

- **Environment:** `C:\workspace\_gladius\arty\.env`
- **Config:** `C:\workspace\_gladius\arty\config.json`
